## Installatie

1. Installeer NodeJS
2. Clone of download deze respository.
3. Open de terminal
4. Voer npm install uit (of npm i) in de terminal
5. Wanneer alles is geinstalleerd, voer npm start uit.
6. De applicatie wordt gehost op http://localhost:8000/
